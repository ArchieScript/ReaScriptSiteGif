RUS:


Запускайте множество действий с помощью одной кнопки и сочетаний клавиш Ctrl, Shift, Alt

Запустите скрипт сочетанием клавиш Ctrl + Shift + Alt + Click в появившемся окне введите id любого экшена или скрипта или часть имени экшена в скобках, а за тем id.

Пример: '[name] id', '(name) id', '{name} id'

Обратите внимание, что в имени недопустимы запятые, если запятые есть, то удалите их, иначе все порушится.

[name name] id - Правильно

[name,name] id - Не Правильно

Для повторного отображения справки, введите в первое поле слово 'help'

Для полного сброса введите в первое поле слово 'reset'









ENG:


Launch multiple actions with a single button and keyboard shortcuts Ctrl, Shift, Alt

Run the script with the keyboard shortcut Ctrl + Shift + All + Click in the window that appears, enter the id of any action or script or part of the name of the action in brackets, and then id.

Example: '[name] id', '(name) id', '{name} id'

Please note that commas are not allowed in the name. If there are commas, delete them, otherwise it will crash.

[name name] id - Correctly

[name,name] id - Not properly

To display the help again, enter the word 'help' in the first field.

For a complete reset, enter the word 'reset' in the first field
