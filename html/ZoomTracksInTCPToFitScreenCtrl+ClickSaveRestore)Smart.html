RUS:


ОПИСАНИЕ ОТСУТСТВУЕТ



Скрипт масштабирует треки в tcp по размеру экрана.

Если надо сохранить высоту треков, то запускаем скрипт с зажатым "ctrl" и высота треков запоминается и загорается кнопка, далее делаем что хотим, также пользуемся этим скриптом, затем опять запускаем скрипт с зажатым "ctrl" и треки восстанавливают ту высоту, которая была на момент сохранения и кнопка гаснет, говоря о том, что сохранения отсутствуют.

А если кнопка уже горит, и надо сохранить, а не восстановить предыдущее сохранение, то запускаем скрипт с зажатыми "ctrl + shift" и скрипт пересохранит высоту треков.

NO DESCRIPTION AVAILABLE



The script scales the tracks in tcp to fit the screen.

If you need to save the height of the tracks, then run the script with the "ctrl" clamped and the height of the tracks is remembered and the button lights up, then do what we want, also use this script, then run the script again with the "ctrl" clamped and the tracks restore the height that was at the time of saving and the button goes off, saying that there are no saves.

And if the button is already lit, and it is necessary to save, not restore the previous save, then run the script with clamped "ctrl + shift" and the script will resave the height of the tracks.
