Archie_View;
Auto enable spectral peaks on selected tracks

Git

Raw







RUS:


Скрипт Автоматически включает спектральное пики на выделенных треках.

Если спектральные пики отображаются некорректно или нужно сбросить (пересканировать) пики то запустите скрипт сочетанием клавиш Ctrl + Shift + Click
чтобы
Сбросить все пиковые файлы кэша

Сбросить Предотвращение всех спектральных пиков

Выключить спектральные пики.

Если нужно восстановить предотвращение спектральных пиков, то запустите скрипт Archie_View; Disabled prevention of spectral peaks spectrogram on all tracks.lua









ENG:


The script automatically includes spectral peaks on selected tracks.

If the spectral peaks are not displayed correctly or you need to reset (rescan) the peaks then run the script by pressing Ctrl + Shift + Click
to
Reset all peak cache files

Reset Prevention of all spectral peaks

Off of the spectral peaks.

If you want to restore the prevention of spectral peaks, run the script Archie_View; Disabled prevention of spectral peaks spectrogram on all tracks.lua
